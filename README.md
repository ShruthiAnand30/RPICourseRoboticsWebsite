# RPI Robotics Documentation Website

This project is a documentation website for RPI robotics kits, designed to make it easier for students to learn, use, and navigate different kits in one place.

The site combines a simple frontend (HTML, CSS, JavaScript) with a lightweight Node.js + Express backend that allows content to be updated directly through an admin (professor) login.

---

## About the Project

This is a collaborative project focused on improving how robotics resources are organized and accessed by students.

The work includes:
- Testing and troubleshooting robotics kits  
- Writing and refining technical documentation  
- Designing and structuring the website interface  
- Building a system that allows content to be updated directly through the website  

The goal is to make robotics documentation more clear, centralized, and easy to maintain over time.

---

## Key Feature: Admin Updates

A core part of this project is allowing instructors to update documentation without needing to edit the codebase.

- Professors can log in through an admin interface  
- Content updates can be made directly on the website  
- Changes are saved and reflected without redeploying the site  
- Reduces the need to manually edit files  

This makes the site more flexible and easier to maintain as kits and instructions change.

---

## Features

- Organized documentation by robotics kit  
- Sidebar navigation with sections and subsections  
- Search functionality for quickly finding content  
- Responsive design for desktop and mobile  
- Backend support for saving and managing updates  

---

## Tech Stack

- HTML, CSS, JavaScript  
- Node.js + Express  
- JSON for lightweight data storage  

---

## How to Run

1. Install dependencies:
   npm install

2. Start the server:
   npm start

3. Open in browser:
   http://localhost:3000

---

## Purpose

Robotics documentation is often scattered or difficult to update. This project aims to centralize resources and provide a system where both students and instructors can easily access and maintain content.
